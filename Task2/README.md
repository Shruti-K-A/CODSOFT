Grade Calculator
A simple yet effective Java-based Grade Calculator that takes marks from multiple subjects, computes the total, average, grade, and displays a personalized feedback message to the user.
It takes the number of subjects and their respective marks as input, calculates:
*Total Marks
*Average Marks
*Grade (A, B, C, D, F)
*A personalized compliment/feedback message based on the grade
#To Compile 
javac Gradecalculator.java
#To Run
java Gradecalculator
