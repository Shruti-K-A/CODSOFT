## Number Guessing Game (Task 1)

This is a simple Java number guessing game where the computer picks a random number between 1 and 100, and the user has 7 chances to guess it. After every guess, the program tells you whether your guess was too high or too low until you get it right (or run out of attempts).

## How to Run

Compile the program:

javac NumberGuessGame1.java


## Run the program:

java NumberGuessGame1

## Input

You just enter a number (1–100).
The program also checks if the input is valid and doesn’t allow non-numeric values.

## Output

Tells you if your guess is too high or too low

Shows the final result (win/lose)

Keeps track of your score

Lets you choose if you want to play again

## Features

Generates a new random number each round

Rejects invalid (non-numeric) input

Exactly 7 attempts per round

Score increases when you guess correctly

Option to play multiple rounds
