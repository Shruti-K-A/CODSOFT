TASK 1 - NUMBER GUESSING GAME
This java program generates a random number in the range of 1 to 100.The user has 7 attempts to guess the number. After each guess, the program tells the user whether the guessed number is too high or too low.
## To compile
javac NumberGuessGame1.java
## To Run 
java NumberGuessGame1
## Input:
User enters integer value.
## Output
Program prints hints(too high or too low) and gives the final results.
#Features
* Random number is generated
* Input validation(Rejects non-numeric input)
* 7 attempts per round
* Score increases when player guesses correctly
* Option to play multiple rounds
